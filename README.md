# kameelkhabaz.github.io

Personal site for Kameel Khabaz. Static HTML + CSS, served by GitHub Pages at <https://kameelkhabaz.github.io/>.

## Edit

- Content lives in `index.html`. Edit the section you want, commit, push.
- Styling is in `style.css`.
- News items are a `<dl class="news">` block — newest at top.
- Publications are an ordered list under `#publications`.
